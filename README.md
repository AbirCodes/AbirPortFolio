# Abir Portfolio Website

This is a fully static portfolio website for Md. Abir Hossain, tailored from CV content.

## Pages

- index.html
- about.html
- services.html
- works.html
- blog.html
- contact.html

## Highlights

- Laravel and backend engineering profile
- AI/ML and data analytics project portfolio
- Direct contact and social links
- No backend runtime required

## Run Locally

Use any static server.

```bash
npx serve .
```

or

```bash
python -m http.server 8080
```
